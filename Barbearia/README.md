# Meireles-Barbearia

Gostaria de deixar registrado que, o nome da pasta principal é diferente do nome da pasta do repositório. Por vezes, ao clonar o repositório, isso gera conflito de interpretação dos caminhos do código. 
Portanto, ao baixar o arquivo, lembre-se de remover a pasta "Barbearia" de dentro da pasta "Meireles-Barbearia", e então abrir a pasta novamente no VSCode. Dessa forma, todos os atalhos funcionarão perfeitamente.
